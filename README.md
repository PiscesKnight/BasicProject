# BasicProject

#### 本项目基于Vue-Cli 3.x，常用的Vue配置，包含如下：
* sass-loader
* node-sass
* sass-resources-loader 全局变量应用

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
